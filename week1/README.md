Generative AI Lab 1 – Synthetic Image Generation

Objective

Generate synthetic images using a pre-trained generative AI model.

Model

Stable Diffusion v1.5 (Hugging Face Diffusers)

Method
	•	Load the Stable Diffusion model in Google Colab
	•	Provide multiple text prompts
	•	Generate images using the diffusion pipeline
	•	Save generated images in the generated_images folder
	•	Display results using Matplotlib

Prompts

Examples:
	•	Futuristic city at sunset
	•	Cute robot reading a book
	•	Mountain landscape with river
	•	Astronaut walking on Mars
	•	Fantasy castle in the clouds

Output

10 synthetic images are generated and stored in the generated_images folder.

Result

A small synthetic image dataset was successfully created using Stable Diffusion based on text prompts.
