 Week 8 – Artistic Image Generation using GAN

Objective:
Generate artistic images using GAN by exploring the latent space.

Model Used:
Deep Convolutional GAN (DCGAN)

Dataset:
CIFAR-10

Tasks Performed:
• Generated 10 artistic images using random latent vectors
• Explored GAN latent space
• Performed interpolation between two latent vectors

Result:
The generator created diverse artistic images and smooth transitions between latent vectors.
